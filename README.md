# Калькулятор ливневого стока по СП

Веб-калькулятор для предварительного расчета поверхностного стока с печатным расчетным листом для пояснительной записки.

## Что реализовано в этом стартовом проекте

- Vite + React + TypeScript.
- Расчетное ядро для годовых дождевых, талых и поливомоечных вод.
- Расчет суточного объема талого стока.
- Расчет расхода дождевых вод методом предельных интенсивностей.
- Расчет объема дождя на очистку.
- Проверка аккумулирующего резервуара.
- Валидатор методических ошибок и предупреждений.
- Пример “Козенки”.
- Печатный лист через CSS `@media print`.
- Заготовка структуры данных источников, мест, метеостанций и коэффициентов.

## Важное замечание по нормативным данным

В проекте есть демонстрационная локальная база. Перед применением в реальном проекте таблицы климатических параметров, коэффициентов и районов нужно заполнить по действующей редакции СП и зафиксировать источник, редакцию и дату проверки.

Сторонние сайты можно использовать только как вспомогательную сверку, а не как единственный источник расчета.

## Запуск

```bash
npm install
npm run dev
```

## Проверка расчетного ядра

```bash
npm test
```

## Публикация на GitHub Pages

1. Собрать проект:

```bash
npm run build
```

2. Опубликовать папку `dist` через GitHub Pages или GitHub Actions.
