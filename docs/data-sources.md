# Источники данных

Основной источник — действующая редакция СП 32.13330.2018 с изменениями №1-5.

В локальной базе должны храниться:

- климатические параметры;
- дождевые районы;
- районы талого стока;
- коэффициенты по типам покрытий;
- диапазонные параметры;
- источник, редакция и дата проверки каждого значения.

Сторонние сайты не должны использоваться как постоянный источник расчета. Их можно использовать только для сверки и ускорения ручного наполнения базы.
