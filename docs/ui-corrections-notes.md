# Правки интерфейса по замечаниям

## Что исправлено

1. Блок места строительства упрощен: больше нет повторяющихся карточек населенного пункта. Альтернативы показываются только по кнопке «Другие варианты».
2. Общая площадь перенесена из блока места строительства в блок покрытий, потому что это контрольная расчетная площадь, а не свойство населенного пункта.
3. Таблица покрытий сделана без горизонтальной прокрутки.
4. Убраны лишние подписи «Шкала» и «Край диапазона — проверьте обоснование».
5. Нормативный диапазон показывается одной короткой строкой под полем, значение выбирается через поле и ползунок.
6. Из видимых признаков покрытий убраны «тверд.» и «снег». Оставлены только прикладные признаки: «Мойка» и «Очистка».
7. Исправлено растягивание карточек, из-за которого появлялись пустые места в блоках параметров.
8. Выравнивание полей ввода сделано плотнее и аккуратнее.

## Что намеренно оставлено

В таблице покрытий оставлены два коэффициента ψ:

- ψ годовой — нужен для годовых объемов дождевого стока;
- ψ расчетный — нужен для расчетного дождя, расхода и очистки.

Это разные расчетные величины, поэтому объединять их в одно поле нельзя без потери методики.
